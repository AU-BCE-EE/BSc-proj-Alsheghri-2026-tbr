here comes the mistake: K2 has units of \(mol/m^3\) but KW has units of \(mol^2/L^2\) so we actually need to correct this. Then the equation becomes \[K_4 = K_2 / (K_W * 10^6) \]

$K_W$ still needs to be in $mol^2/L^2$ because of the specaition model.  
All the equlibrium constants passed to the speciation model needs to be in $mol/L$ units.
And all the constants used in the main model and the enhancement factor model needs to be in $mol/m^3$
